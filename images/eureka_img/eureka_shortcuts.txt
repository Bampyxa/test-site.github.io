Клавиатура и мышь.

Общие комбинации для всех режимов:

LMB(ЛКМ)
 выберите объект, перетащите, чтобы переместить объект(ы)
 щелкните в пустой области, чтобы очистить выбор
 щелкните в пустой области и перетащите, чтобы выбрать группу объектов

RMB(ПКМ)
 перетащить карту

MMB(СКМ)
 вставить объект (так же, как SPASE или INSERT)
 с SHIFT: масштабировать выбранные объекты
 с CTRL: вращать объекты
 колесо мыши: увеличить или уменьшить масштаб

1..9 : выберите размер сетки (от меньшего к большему)
TAB : переключать между 3D-режимом и режимом редактирования
; : сделать следующей нажатой клавишей - META

t : включить режим расстановки вещей, монстров, оружия и др.
l : режим редактирования линий
s : режим редактирования секторов
v : режим редактирования вершин
b : переключение бокового меню

CTRL-Z : отменить(может быть использована несколько раз)
CTRL-Y : отменить предыдущую отмену
CTRL-A : выбрать всё на карте
CTRL-I : инвертировать выделение
CTRL-U : очистить выделение(так же кнопка `)

HOME : поместить карту посередине экрана
END : поместить 3D-камеру посередине экрана
' (quote) : переместить 3D-камеру в положение указателя мыши

META-N : загрузить следующий файл из списка
META-P : загрузить предыдущий файл из списка
N : открыть следующую карту в текущем wad-е
P : открыть предыдущую карту в текущем wad-е

g : регулировка размера сетки: меньше
G : регулировка размера сетки: больше
h : режим переключения сетки : вкл./ простой вид /выкл.
f : режим переключения прилипания к сетке: вкл. или выкл.
J : показать/скрыть номера всех объектов
j : перейти к объекту номер...
o : скопировать и вставить выбранные объекты
c : скопировать свойства из выделенного объекта на выбранный мышью
p : подрезать неиспользуемые секторы, линии и вершины
H : "отзеркалить" объект по горизонтали
V : "отзеркалить" объект по вертикали
R : повернуть объект на 90 градусов по часовой стрелке
W : повернуть объект на 90 градусов против часовой стрелки
\ : показать/скрыть ранее использованное в боковом окне
META-F : применить новый тег в текущих объектах
META-L : применить последний (самый высокий) тег в текущих объектах


Things Mode(Режим вещей):
SPACE : добавить новую вещь
d : если вещи находятся в одной точке, разложить их на некотором удалении
m : поставить рядом стоящие вещи в одну точку
w : повернуть вещь на 45 градусов против часовой стрелки
x : повернуть вещь на 45 градусов по часовой стрелке


Vertex Mode(Режим вершин):
SPACE: добавить новую вершину, если одна уже выбрана, рисует линию
d : отсоедините все линии в выбранных вершинах
m : слить выбранные вершины в одну(в общем центре)
I : выстроить выбранные вершины в одну линию
O : выстроить выбранные вершины в круг
D : выстроить выбранные вершины в полукруг
C : выстроить выбранные вершины по 120-градусной дуге
Q : выстроить выбранные вершины по 240-градусной дуге


Linedef Mode(Режим линий или точнее плоскостей с видом сверху):
d : отсоединить выбранные линии от остальных
m : объединить две односторонних линии в двустороннюю
e : выделить цепочку линий, уже выделив одну и наведя мышь на другую
E : выделить цепочку линий с такими же текстурами
w : развернуть линию в другую сторону
k : разделить линию пополам


Sector Mode(Режим секторов):
SPACE: добавить новый сектор в точке указателя мыши или исправить сломаную разметку в области (используйте клавишу CTRL)
d : отсоединить сектор(ы) от соседних
m : объединить все выбранные сектора в один
e : выберите сектора с той же высотой пола
E : выберите сектора с одинаковой текстурой пола
w : поменять в выбранных секторах напольные и потолочные текстуры местами
, and < : понизить высоту пола в секторе
. and > : повысить высоту пола в секторе
[ and { : понизить высоту потолка в секторе
] and } : повысить высоту потолка в секторе


3D Preview(3D-режим):
Используйте для управления камерой в 3D-режиме кнопки со стрелками или кнопки WASD
LMB : не реализовано
RMB : поворот камеры при движении мыши
MMB : настройка смещения текстур на выделенной стене(перетащить мышью)
wheel : перемещать камеру вперед или назад
PGUP and PGDN : перемещать камеру вверх или вниз
g : переключатель гравитации: режим полета или режим ходьбы
l : переключатель освещения вкл. или откл.
t : переключатель текстур: показать или скрыть
o : переключатель вещей: показать или скрыть
F11 : увеличение яркости (гаммы)
c : выровнять подсвеченную мышью текстуру(автоматически)
x : выровнять по оси X со стеной слева(начинать лучше с места разрыва текстур-другой текстуры)
y : выровнять по оси Y со стеной слева
z : выровнять по обоим осям со стеной слева
X : выровнять по оси X со стеной справа
Y : выровнять по оси Y со стеной справа
Z : выровнять по обоим осям со стеной справа
